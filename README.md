# Prospecção Fria

Plataforma unificada para gerenciamento de atendimentos no WhatsApp, suporte em equipe e prospecção automatizada.

## Principais Funcionalidades
- **Interface Premium**: Design moderno com foco em UX e clareza.
- **Integração WAHA**: Gestão robusta de sessões de WhatsApp.
- **Chat em Tempo Real**: Conversas fluidas com indicadores de status e entrega.
- **Painel de Controle**: Gestão de leads e campanhas.

## Tecnologias
- **Frontend**: React, Lucide Icons, Vanilla CSS.
- **Backend**: Node.js, WAHA API.
- **Banco de Dados**: Supabase (PostgreSQL).
